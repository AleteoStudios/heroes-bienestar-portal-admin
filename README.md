# Héroes del Bienestar - Portal Administrador

Portal web para consultar perfiles infantiles e historial de mediciones mediante código QR.

## Funciones actuales

- Lectura de QR desde URL.
- Búsqueda manual por código QR.
- Consulta de perfil desde Supabase.
- Consulta de historial de mediciones desde la vista `vw_profile_health_history`.

## Tecnologías

- HTML
- CSS
- JavaScript
- Supabase JS

## Estado

Prototipo funcional conectado a Supabase.
